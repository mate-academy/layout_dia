# Dia landing page
Implement landing page according to [Figma design](https://www.figma.com/file/7qwsWggv9BAxMi2VPhBuPr/Air-(formerly-Dia)?node-id=9138%3A35) - Use BEM and SCSS

Markup & style: HTML, CSS (Grid/Flexbox)
Methodology & preprocessing: BEM, SASS (SCSS)
Adaptive design(desktop, tablet, mobile)

### Installing
* Fork and clone this repository
* Run npm install in your terminal
* Run npm start

Adapt page to the following screens:
- Large screens 2560px
- Full HD 1920px
- The design 1600px
- Notebook 1280px
- Tablet 1024
- Mobile (> 320px)
